Fusion of visible and infrared images via saliency detection using two-scale image decomposition

Publication Information

  • Role: Co-Author , Student Lead
  • Company: IJST, Springer Publications
  • This project proposes a fusion method that develops a weight map procedure which can viably recognize and recreate source picture data after fusion of Visible and IR images.
  • Comparative analysis is done by considering Entropy, Standard Deviation, Structural similarity Index (SSIM) and Peak signal-to-noise ratio (PSNR) of the images.
  • Used Deep learning Image processing model like Convolutional neural Network (CNN) and several other Machine Learning techniques like Laplacian Pyramid, Sparse Representation and Adaptive Sparse Representation.
